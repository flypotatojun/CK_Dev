第三方mel程序添加方法：

可根据 F:\tjh_pipeline_tools_dll\scripts\common 下的sample.mel 例程规范参考。


1、添加程序 及 菜单显示：
sample.mel 
此工具集中收集了Maya各模块中常用的各种mel工具，模块具备自动更新功能，请将要添加的mel
,mel中不能包含中文名，格式< melName.mel >拷贝到对应分类的文件夹中，并重启maya更新列表。

2、添加程序 及 菜单不显示：

mel中不能包含中文名，格式< sample.otherName.mel >，此程序会被source，但在菜单不显示。

3、屏蔽程序：

注意:移除程序中的UI生成的执行语句；如果去除模块中特定的程序，可以将其重命名".bak"文件即可。

4、帮助添加：

每个工具右侧的工具选项为对应工具的帮助信息。可添加同名.txt文件及可，也可在UI中修改，添加sample.txt 。

快速查找命令，

5、查找程序；

可以通过主菜单的 Quick search command 来实现，输入要找命令包含的字符，回车即可快速显示。

模块内功能python程序添加方法：
===================================================================

1、将python插件*.py 或是整个模块目录拷贝到对应的分类文件夹中。

sample.mel //同名调用脚本
sample.txt //同名帮助文档
sample.py //原始py脚本
<tjh_test> //python模块整个包目录

2、编写同名tjh_test.mel调用脚本。
建立一个同名的mel脚本，其中输入如下调用代码：

global proc tjh_test()
{
	python("import tjh_test");//导入模块
	python("tjh_test.winUI()");//调用UI界面主程序
}
