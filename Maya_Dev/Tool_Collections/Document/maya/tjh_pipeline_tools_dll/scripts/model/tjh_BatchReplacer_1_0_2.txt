2、使用说明：

 

根据步骤提示，进行操作即可。

1）点击 create nPartciles...按钮，激活n粒子创建工具，之后在场景中点击你要大量摆放物体的位置，做好前期布局工作，比如在场景中大量布置树木的位置。石块的位置等。最后，按回车确定，创建n粒子位置点。

 

2）之后，选择你要在粒子位置替换的种子物体，注意这里支持多选。比如你希望使用的多种树木或是石块等。点击select replace seeds 按钮确认。

 

3）选择之前创建的n粒子，点击select objects will be replaced按钮。确定要被替换的物体。

 

4）点击replace now 按钮，执行替代过程。注意：替换时，有两种方式，copy物体方式，还有instance实例方式。通过设定step per 的值，可以进行隔步替代。比如你不想替换所有的物体，而是想，隔一个替换一个，此时可以设置step per 为2，以此类推。

 

5）替换完的大量物体，可以通过面板下面的“随机位移、旋转、缩放、拓展间距工具“进行后期的快速随机化处理。其中拓展间距工具比较有趣，可以后期批量调节物体间的间距，使它们变的疏散和稠密。




3、参数说明：

 

一）批量替代工具参数说明：

select replace seeds：选择用于替代的种子物体，可以是多边形或nurbs物体。

selcct objects will be repalced：选择被替代的物体，可以是手动创建的粒子、或是多边形上的顶点，或是独立的其他物体。

copy：使用复制方式替换物体。

instance：使用实例化方式替换物体。

delete the orig replaced objects选项：勾选此选项，删除原始别替代物体。

Maintain group level relationships 选项（1.0.2版新功能）：保持被替代物体组级关系。 
step per ：可根据提供的步数跳跃替换选择物体。此方式可以实现隔物体，跳跃式替换效果。

replace now：执行替代操作。

Undo（1.0.2版新功能）：撤销操作。

 

二）随机位移、缩放、旋转、拓展工具参数说明：

选择要操作的物体，填写参数，点击相应按钮，即可。

 Lock x y z（1.0.2版新功能）： 锁定xyz值，结果如x = y = z .

Move：执行x、y、z指定距离内的随机位移。

Rotate rand ： 指定角度内，随机Y轴方向旋转。

Scale rand : 指定缩放比例内，随机xyz方向缩放。

Center Pivot for all selects：批量重置物体局部轴中心位置。

Spread space：对批量选择物体执行向四周扩散位移操作。
