1. Copy the folder to any path (the path cannot contain Chinese)

2, use the editor to open the bat file to modify the file path (PATH is python.exe) using cgteamwork's python interpreter can not be modified

3. Open the mainWindows file and find the MAYABATCHPATH modify path to the local maya path or service its maya path.

4. Run the run.bat file (an interface will be displayed if the above operation is correct)

5. Drag the Maya animation file into the interface and click the OK button to run the program.

6. Finally, the exported animation curve file will appear in the same folder as the dragged file.
